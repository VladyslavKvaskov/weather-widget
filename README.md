# weather-widget
A simple, dynamic weather widget SVG built with Next.js, leveraging free Vercel hosting and the OpenWeather API. It accepts the same query parameters as OpenWeather's Current Weather API. Additionally, the widget supports a viewBox parameter to customize the SVG's viewBox attribute. Pass four space-separated numbers, e.g., `?viewBox=0 0 180 130`.
